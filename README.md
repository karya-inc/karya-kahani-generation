# Karya-Kahani-Pipeline

Multi-Agentic culturally nuanced educational content generation framework

This framework addresses the need for accessible education in rural India by creating engaging, story-based content. It consists of two main modules:

1. **Story Generation Module** - Creates 5-module curricula through compelling narratives
2. **Image Generation Module** - uses FLUX model to generate illustrations in Amar Chitra Katha style


## Features

### Story Generation
- **5-Module Curriculum**: Progressive concepts
- **Cultural Context**: Tailored for low-income Indians in specific regions
- **Narrative Structure**: Soap opera-like drama with educational content
- **Character Development**: Well-developed protagonist journey
- **Azure OpenAI Integration**: Uses GPT-4 for content generation

### Image Generation
- **FLUX Model Fine-tuning**: Custom training for Amar Chitra Katha style
- **LoRA Training**: Efficient fine-tuning with reduced computational requirements
- **Multi-resolution Support**: 512x512, 768x768, and 1024x1024 outputs
- **Cultural Art Style**: Authentic Indian comic book aesthetics
